## 1.开通券商账户
中资、港资券商不要再开了，包括但不限于富途、老虎、长桥、华盛、盈立。一个是门槛变高，开户需要提供一些海外身份证明材料才能开，或者花钱找中介。更主要的是CRS问题，你的所有股票交易信息、账户余额都会同步给国家税务局。  
那么该怎么办？  
开美资券商，总所周知，美国是唯一一个不向别的国家同步税务信息并要求其他国家向他同步税务信息的大国，其他的包括新加坡、英国都不行。
