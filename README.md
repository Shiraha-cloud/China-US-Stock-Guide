## 一.开通券商账户
### 不推荐中资或香港本土券商  
一个是门槛变高，开户需要提供一些海外身份证明材料才能开，或者花钱找中介。更主要的是CRS问题，你的所有股票交易信息、账户余额都会同步给国家税务局。老虎、富途、长桥、盈立、华盛、辉立、耀才都属于中资或香港本土券商。  
  
众所周知，美国是唯一一个不向别的国家同步税务信息并要求其他国家向他同步税务信息的大国，其他的包括新加坡、英国都不行。

## 二.开通银行账户
经过第一步，你就已经拥有一个合规的美资券商账号了。
现在你要开通一个海外的银行账号，常见的港卡有中银香港、汇丰、众安、蚂蚁，这些开的人最多，前俩为实体银行，后俩为虚拟银行。从26.5.22开始港卡开通在收紧，中银香港已经有很多人反馈线下也开不了了，其他三个也在逐步收紧，建议没有办的朋友也可以赶最后窗口上车，只需定一张去香港的机票即可，车门一旦焊死，可能永远也不会打开。    
以上这些港卡是合法合规的银行，合法合规意味着一定会参与CRS信息交换，于是就有小伙伴担心被CRS问题了。  
那么怎么办？    
开美卡，美国银行卡。前面已经说过了，美国不参与CRS交换，券商和银行都不参与。

如果资金量不高，建议直接上华美，毕竟开卡和持有成本都比较低。如果是大户，还是找个机会去一次美国，求个安心。  
我自己开通了华美，目前使用正常，华美银联卡可以直接绑定微信支持宝用于国内消费，美元自动转换，汇率我看了还不错。  

如果以上银行都不想开，还有一种选择是Wise。Wise是一个虚拟银行，总部在英国，它的主要作用就是提供一个全币种账户（人民币、美元、港币、欧元、英镑），可以接收香港卡的港币或美元转账，也能接收国内银行的转账（建议先购汇英镑再转），也可以绑定支付宝、微信提现。开户只需要提供身份证即可，需要注意的是开通后需要发邮件联系客服将中文名字改为英文名字（张三->ZHANG SAN）。欢迎通过<a href="https://wise.com/invite/ilpc/c2qg6k6" target="_blank">我的Wise邀请链接</a> 注册，可享受首笔跨境转账免手续费优惠。注册和登录 Wise 时，尽量直接使用中国大陆原生内网。



## 三.资金进出

经过第一步和第二步，你拥有了合规的**美资券商账户**和**美国银行卡**。  
接下来我们开始聊资金进出：
众所周知，每个人每年有5W美金外汇额度，我们来聊聊入金和出金。

### 入金（资金由银行到券商账户）
标准流程下一般是这样：**国内银行购汇->香港卡->美资券商**。国内银行到香港卡一般可通跨境汇款或跨境支付通来进行，二者都占用额度，区别是跨境支付通比较适合小额，且仅支持港币和人民币。

当然，如果你没有香港卡，你也可以借助wish来入金，流程大概是这样：**国内银行购汇后（推荐英镑）->Wise账号->券商**。

经过多方验证，最佳省钱的方式是：**国内兴业寰宇卡->汇丰港币>Wise换美元->美国银行卡->美资券商**，基本可以做到无损入金。

此处提一下，兴业寰宇卡是个汇率神卡，推荐可以办都办一张。


### 出金（资金由券商账号到银行）
标准流程下一般是这样：**美资券商->香港卡->国内银行结汇**。这种方式好处是稳定，直接坏处是CRS。  

当前，如果你不想CRS，也可以这样：**美资券商->美国银行卡**。到此就截止了，不往后了。这种方式好处是不参与CRS，但缺点是不稳定。如果你持有的是小型银行，一旦银行因为一些奇怪的原因把你卡冻结了，处理起来比较麻烦。  

美国卡可用于日常消费，比如绑定微信支付宝直接在国内使用，一年大概能用6w人民币，不占用外汇额度。也可以在booking或者trip直接购买国内的机票酒店，在国内直接消费。你还可以购买一些海外游戏卡之类的。有很多可以花的地方。  



## 四.FAQ（常见问题）

**Q：大陆居民没有境外银行卡，可以直接开户炒美股吗？**
A：可以。先开通支持大陆护照的嘉信理财（Charles Schwab）或第一证券（Firstrade），后续配合华美 Velo 或 Wise 进行入金。该方式不稳定，但有些国内银行可以成功。

**Q：大陆居民炒美股要交美国税吗？**
A：中国大陆居民属于非美国居民（Non-resident Alien），开户时券商会让你确认 W-8BEN 表格。这意味着你不需要向美国政府缴纳任何股票买卖差价的资本利得税，只有美股分红（股息）会被美国扣除 10% 的预扣税（中美税务互惠协定），非常划算。



**Q：如何实现无损入金？**
A：最佳省钱路径为：国内兴业寰宇卡 -> 汇丰港币 -> Wise 兑换美元 -> 美国银行卡 -> 美资券商。

**Q：如何降低美国银行卡和 Wise 被冻结的风险？**
A：严格坚持“同名账户转账”（即自己转给自己）。在 Wise、银行、券商之间流转资金时，确保姓名拼音（如 ZHANG SAN）完全一致。
